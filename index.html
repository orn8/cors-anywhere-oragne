CORS Proxy

To use the CORS proxy, use the following URL structure:

https://oragne-proxies.vercel.app/api/cors?url=[your url]

Replace [your url] with the URL you want to fetch, and it will bypass CORS restrictions.

Example: https://oragne-proxies.vercel.app/api/cors?url=https://example.com

Ad Removal Proxy

To remove ads from a page, use the following URL structure:

https://oragne-proxies.vercel.app/api/adless?url=[your url]

Replace [your url] with the URL of the page you wish to remove ads from.

Example: https://oragne-proxies.vercel.app/api/adless?url=https://example.com

Tip: If you're encountering errors, try encoding your replaced URL using HTML URL encoding. You can use an online tool like this URL encoder to ensure proper formatting.
